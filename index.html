Website of Kantvikin Tulentekijät ry

I created a simple Wordpress site for our local scout troop, Kantvikin Tulentekijät. The process included defining the main users, their goals, collecting the content and customising the CSS of existing theme. In addition I configured automated backups for a Dropbox account and payed attention to SEO.

User interfaces and usability

At this course we learned about user centered design process and applied the knowledge in a group work. Our group choose Aalto Sharetribe. We evaluated the usability of the service using Nielsen’s heuristics, interviewed four users of the service and finally created a new feature, advanced search with notifications, to fulfil the goals of the users. The new feature was demonstrated using storyboards and Lo-Fi prototypes.

Web development and digital media

During Studio 2 course I learned about HTML, CSS, JavaScript and Photoshop. We created a visual campaign with a poster, website and a game for consolidation of municipalities.

Java project - A 2D platform game

This project was the final exercise at Studio 1 programming course for freshmen. The beginning was development of ideas, after that choosing Slick2D library to use and planning the MVC architecture of the game. During the project I learned about software development, working independently and using external libraries. The visuals used in the game are drawn by Sonja Kiiveri
